# Agrinho-2026

# 🌱 Projeto MIP - Manejo Integrado de Pragas

> Repositório oficial estruturado para o **Concurso Agrinho 2026** 🏆
> Hashtag do projeto: `#agrinho2026`

---

## 📝 Objetivo Alinhado ao Tema

Este aplicativo web foi desenvolvido sob as diretrizes do tema oficial do ano: **"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**. 

Seu objetivo é instruir de forma interativa sobre o **Manejo Integrado de Pragas (MIP)**, mostrando como pequenos e grandes produtores podem monitorar ameaças agrícolas e tomar decisões sem recorrer imediatamente ao uso indiscriminado de defensivos químicos.

---

## 🛠️ Tecnologias Utilizadas

O ecossistema foi modularizado para máxima pontuação na rubrica de avaliação:

- **HTML5:** Tags semânticas limpas organizadas no padrão mobile-first.
- **CSS3 Personalizado:** Variáveis nativas (`:root`), cantos acentuadamente arredondados simulando interfaces nativas de apps modernos e paleta composta por variações de verdes e azuis.
- **JavaScript Moderno:** Manipulação assíncrona do DOM via eventos orientados ao carregamento da janela, impedindo congelamento de scripts.

---

## 🚀 Critérios da Rubrica Preenchidos

1. **Complexidade Funcional:** Chat responsivo que sugere trocas inteligentes de métodos agroecológicos baseado no feedback do usuário.
2. **Acessibilidade:** Escalonamento dinâmico do tamanho das fontes do documento via funções do mecanismo JavaScript.
3. **Organização Arquitetural:** Separação rígida de escopos corporativos em arquivos `.html`, `.css` e `.js`.
